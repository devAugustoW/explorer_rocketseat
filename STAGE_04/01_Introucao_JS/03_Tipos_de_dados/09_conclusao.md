# Tipos de dados

Conforme o ECMAScript Standart temos 9 tpos de dados:

* Data types
    * Primitive / Primitive value
    * Struvtural
    * Strutural Primitive

## Primitivos

    * String
    * Number
    * Boolean
    * Undefined
    * Symbol
    * BigInt

## Estruturais

    * Object
        - Array
        - Map
        - Set
        - Date
        - ...

## Primitivo Estrutural / Structural Root Primitive

    * Null
